# Commands

```bash
# initialises in the current directory
lcdl init 
# downloads a range of leetcode problems
lcdl get-range 100 200
# downloads the problem with the given numbers
lcdl get 1 2 3
# downloads problem with the specific url
lcdl get-url https://leetcode.com/problems/two-sum/
# Checks the latest solution you've completed (can be done offline)
lcdl check
# Checks the problems with numbers
lcdl check 1 2 3
#Checks all problems
lcdl check --all
# Submits the most recent solution (requires internet)
lcdl submit
# Submits the solution with the numbers 
lcdl submit 2 3 4
```
